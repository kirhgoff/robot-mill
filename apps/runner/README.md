# runner

Environment variables:
- `TASK_DIR` (default `/data/tasks`)
- `WORKSPACES_DIR` (default `/data/workspaces`)
- `ROBOT_ID` (default `runner-1`)
- `ROBOT_PLAN_COMMAND` (shell command reading JSON from stdin and outputting JSON)
- `ROBOT_IMPLEMENT_COMMAND` (shell command reading JSON from stdin and outputting JSON)
- `SLEEP_MS` (default `2000`)
